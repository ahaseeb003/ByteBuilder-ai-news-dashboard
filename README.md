# AI Tech News Dashboard v3

> **Developed by HMtechie & ByteBuilder**

Real-time AI & Tech news aggregation, research paper discovery, and channel-ready post generation powered by LangGraph multi-agent pipelines.

## Two-Panel Architecture

| Panel | Purpose |
|---|---|
| 📡 Trending News Feed | Browse latest articles, research papers, and GitHub repos. Click Add to Post Creator on any item. |
| ✍️ Post Creator | Choose a template, select content, and generate a channel-ready post with one click. |

## Features

- 35+ live RSS feeds (OpenAI, DeepMind, TechCrunch, arXiv, etc.)
- Research Papers from arXiv + Papers With Code
- GitHub Trending with Top 3 podium
- 4 post templates: Quick News Flash, Tech Insight, Research Paper, Weekly Update
- Real-time research post generator (keyword / URL / arXiv ID)
- ByteBuilder follow footer on every post
- Reference links on every post
- LangGraph multi-agent pipeline
